### Key Features

- Add and delete your todos
- Mark as completed
- Filter todos: all, completed, uncompleted
- Display statistics of your task: total count, completed, uncompleted, percentage of completed
- Delete one by one or clear all
- Random quote for motivation at the top
- Recoil state manager
- Chakra UI components
- toast for errors



## Tech
- Javascript
- Vite
- React
- Chakra UI
- Recoil

## Getting Started

### Cloning the repository

```shell
git clone https://github.com/otomilova/todoApp_recoil.git
```

### Install packages

```shell
npm i
```

### Start the app

```shell
npm run dev
```
